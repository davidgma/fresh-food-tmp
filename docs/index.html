Node, TypeScript, Sass Boilerplate

The purpose of this StackBlitz project is to provide a starting point for a simple project using HTML, Sass and TypeScript. The standard one provided by StackBlitz doesn't include sass ðŸ˜Ÿ.

All the working files are under the 'docs' folder. This is so that if it is saved to Github it can be easily served by Github Pages as a website.

Any .scss files in the docs folder will be watched by sass and compiled into styles/styles.css.

Any .ts files in the docs folder will be watched by TypeScript and compiled into the javascript folder.

In development, Vite will serve the web page and watch for any changes to the input files.
